# Klee Spark Knight Special Site

A single-page character special site built with `Vue 3 + Vite`, centered on Klee from Genshin Impact.

Live site:

- [https://klee727.cn/](https://klee727.cn/)

## What is inside

- Klee hero landing section
- Character profile and impression modules
- Local gallery assets
- Local voice clips with inline audio players
- Event memory timeline with local cover images
- GitHub Pages static deployment

## Tech stack

- Vue 3
- Vite
- Native CSS
- GitHub Pages

## Local development

Install dependencies:

```bash
pnpm install
```

Start dev server:

```bash
pnpm dev
```

Build static output:

```bash
pnpm build
```

## Project layout

- `src/`: page and component source
- `public/`: local images, audio, and other static assets
- `docs/`: GitHub Pages publish output

## Deployment

The repository is deployed through GitHub Pages with the custom domain `klee727.cn`.
The current publish source is `main /docs`.
